# Prompt Templates

## Interior Page Prompt

```
Create an illustration for a children's storybook page.
Use the locked Story Bible and character reference exactly.

Book title: [BOOK_TITLE]
Target age: [TARGET_AGE_RANGE]
Illustration style: [LOCKED_ILLUSTRATION_STYLE]
Color palette: [LOCKED_COLOR_PALETTE]
Line art style: [LOCKED_LINE_ART_STYLE]
Rendering style: [LOCKED_RENDERING_STYLE]

Page number: [PAGE_NUMBER]
Scene: [SCENE_SUMMARY]
Characters present: [CHARACTERS_PRESENT]
Main character appearance lock: [EXACT_CHARACTER_DESCRIPTION]
Character outfit lock: [EXACT_OUTFIT]
Character expression: [EXPRESSION]
Setting: [LOCATION]
Mood: [EMOTIONAL_TONE]
Continuity notes: [CONTINUITY_NOTES]

Important:
- The character must look exactly like the same character from the reference sheet.
- Do not change face, body shape, outfit, colors, age, species, accessories, or style.
- Do not change the illustration style.
- Do not introduce random new visual elements.
- The image must look like it belongs to the same children's book as every other page.
- Absolutely no text, letters, numbers, words, logos, or typography inside the image.
```

## Cover Prompt

```
Create a premium children's storybook cover.
Use the locked Story Bible and character reference exactly.

Book title: [BOOK_TITLE]
Target age: [TARGET_AGE_RANGE]
Story theme: [STORY_THEME]
Main character: [MAIN_CHARACTER_NAME]
Main character exact appearance: [EXACT_CHARACTER_DESCRIPTION]
Character outfit: [EXACT_OUTFIT]
World setting: [WORLD_SETTING]
Illustration style: [LOCKED_ILLUSTRATION_STYLE]
Color palette: [LOCKED_COLOR_PALETTE]
Line art: [LOCKED_LINE_ART_STYLE]
Cover mood: [COVER_MOOD]
Cover scene: [COVER_SCENE]

Important:
- The cover character must match the interior character exactly.
- Same face, body shape, colors, outfit, line style, rendering style, and world setting.
- The cover must look like the same book as the interior pages.
- Do not redesign the character. Do not change the art style.
- Absolutely no text, letters, numbers, words, logos, or typography inside the image
  (title/subtitle will be typeset separately by the app).
```

## Character Reference Sheet Prompt

```
Create a character reference sheet for a children's storybook character.
No background scene — neutral cream backdrop, orthographic art-book style.

Character name: [CHARACTER_NAME]
Species/type: [SPECIES]
Exact appearance: [EXACT_CHARACTER_DESCRIPTION]
Outfit: [EXACT_OUTFIT]
Signature accessory: [SIGNATURE_ACCESSORY]
Palette: [COLOR_PALETTE_HEX]
Illustration style: [LOCKED_ILLUSTRATION_STYLE]
Line art: [LOCKED_LINE_ART_STYLE]
Rendering: [LOCKED_RENDERING_STYLE]

Show, in one composed sheet:
- front full-body view
- 3/4 view
- side view
- three facial expressions (happy, curious, sad)
- close-up of signature accessory
- palette swatches along the bottom

Important:
- Absolutely no text, letters, numbers, or words in the image.
- Same character across every pose — do not redesign between views.
```
