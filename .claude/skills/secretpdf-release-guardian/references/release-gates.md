# Release Gates

These are minimums. Do not reduce them to pass a release.

## Hard zero-count gates

- blank or placeholder cover: 0
- duplicate logical pages: 0
- duplicate normalized text blocks: 0
- unintended duplicate image hashes: 0
- raw Markdown artifacts: 0
- HTML comments/internal page keys: 0
- watermarks/signatures/URLs in illustrations: 0
- unapproved random image text: 0
- truncated or incomplete text blocks: 0
- metadata mismatches: 0
- unverified public claims: 0

## Minimum scores

- character identity consistency: 95
- cover-to-interior match: 95
- illustration style consistency: 95
- page-to-page continuity: 95
- text-to-image semantic match: 95
- story chronology: 98
- age appropriateness: 95
- typography/layout: 95
- cover quality: 90
- thumbnail/product mockup quality: 90
- sales-page sanitization: 100
- product metadata match: 100
- final sellable quality: 92

## Permanent-fix proof

- original failing fixture passed: true
- consecutive fresh books passed: at least 3
- manual DB edits: 0
- threshold reductions: 0
- gate bypasses: 0
- clean install/typecheck/tests/build: all true
