# Project Memory

## Core
Kids picture books: 32 pages standard, 8.5×8.5", watercolor+ink storybook style, character invariants locked per page.
Kids cover title MUST be a custom illustrated LOGO designed for that book (Peppa Pig / Bluey / Paddington / Gruffalo tier) — painted in same medium as illustration, every letter uniquely hand-drawn with story-linked decoration (vines/berries/leaves/ears/stars). NEVER an existing font, NEVER hand-drawn fonts, NEVER flat text overlay. Emotion words must visually distort (Wobbly=wavy, Sleepy=droops, Zoom=streaks).
User does not use Shopify anymore — skip all shopify- steps, publish natively via storefront + Stripe.
Auto-price kids picture book: base $6.99 + $2 if ≥32pg + $1 if cover_score≥90 + $1 if original character bible → round to .99.
